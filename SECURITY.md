# Security Policy

## Supported Versions

| Version | Supported |
| --- | --- |
| 0.2.x | Yes |
| < 0.2.0 | No |

## Reporting a Vulnerability

Please report security issues privately by opening a GitHub Security Advisory draft:
https://github.com/gaestu/surfsifter/security/advisories/new

Please include:
- A clear description of the issue and impact.
- Reproduction steps or a proof-of-concept.
- Affected version(s) and environment details.

## Response Targets

- Initial acknowledgement: within 3 business days.
- Triage and severity assessment: within 7 business days.
- Remediation plan or timeline: within 14 business days.

We will coordinate disclosure timing with reporters and request reasonable time to ship a fix before public disclosure.
