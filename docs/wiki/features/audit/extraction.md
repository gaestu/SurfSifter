# Extraction (Subtab)

View all files extracted by any extractor (extracted_files table)

## Purpose
- Extraction subtab - displays extracted_files table.
- Shows all files extracted by any extractor with filters and summary stats.

## Inputs
- Evidence database extracted_files table and related extractor metadata.

## Filters and controls
- Filter by extractor name
- Filter by status
- Summary statistics
- Paginated table view

## Outputs
- Summary statistics and paginated table view (read-only).

## Notes
- Use Refresh after running extractors to load new entries.
