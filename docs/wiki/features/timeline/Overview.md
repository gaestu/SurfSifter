# Timeline (Tab)

Unified, filterable timeline built from multiple extracted artifacts.

## Purpose
- Combine events from many sources into a single investigative timeline.
- Support filtering, tagging, and export for reporting.

## When to use
- After ingestion to build a cross-artifact time view.
- When you need to export or tag timeline events for analysis.

## Data sources
- Evidence database timeline table (built from artifact sources).
- Source artifacts such as history, downloads, cookies, bookmarks, sessions, and jump lists.

## Key controls
- Build Timeline button (generates the unified timeline).
- Filters: event type, confidence, tag, and date range.
- Actions: Tag Selected, Export CSV, Refresh, pagination controls.

## Outputs
- Timeline events stored in the evidence database.
- CSV export of the filtered timeline view.

## Subtabs
- None

## Notes
- Build Timeline must be run at least once to populate data.
