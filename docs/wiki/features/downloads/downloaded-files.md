# Downloaded Files (Subtab)

Downloaded Files subtab - view downloaded non-image files.

## Purpose

- Review downloaded non-image files with key metadata.
- Tag, export, and open downloaded files during analysis.

## Inputs

- Case database downloads table (completed non-image downloads).
- Case downloads folder (stored files).

## Filters and controls

- Filters: file type, domain, and filename search.
- Table view with context menu (open file/folder, copy URL/hash).
- Actions: Tag Selected, Export to Folder, View Properties, Open Folder.

## Outputs

- Exported file copies to a chosen folder.
- Tags written to the case database for selected downloads.
- Properties dialog for quick review of a file's metadata.
