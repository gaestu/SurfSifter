# Downloaded Images (Subtab)

Downloaded Images subtab - view downloaded image files with thumbnails.

## Purpose
- Review downloaded image files with thumbnails.
- Tag, export, and analyze images (hash checks and similarity).

## Inputs
- Case database downloads table (completed image downloads).
- Case downloads folder and thumbnail cache.

## Filters and controls
- Filters: domain and filename search.
- Thumbnail grid with multi-select (double-click to open).
- Actions: Tag Selected, Export, Check Hashes, Find Similar, Open Folder.

## Outputs
- Exported image copies to a chosen folder.
- Tags written to the case database for selected downloads.
- Hash check and similarity results shown in dialogs; thumbnails cached on disk.
