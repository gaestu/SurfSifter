# Available Downloads (Subtab)

Available Downloads subtab - browse and select URLs for download.

## Purpose

- Review URLs that are available to download from extracted evidence.
- Select and queue URLs for download with filtering and pagination.

## Inputs

- Case database URL tables (including source and match/tag metadata).
- URL match lists and tags for filtering.

## Filters and controls

- Filters: domain, tag, match status/list, type, and URL text search.
- Pagination controls and page size selection.
- Selection actions: Select All/None and Download Selected.

## Outputs

- Download requests queued for the Downloads workflow.
- Download status recorded in the case database.
