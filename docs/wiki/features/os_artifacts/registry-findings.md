# Registry Findings (Subtab)

Registry Findings subtab - extracted registry indicators.

## Purpose
- Review OS registry indicators linked to browser activity.
- Filter by indicator type to narrow findings.

## Inputs
- OS indicators table in the evidence database.

## Filters and controls
- Filter: indicator type.
- Export CSV button.
- Table view with sortable columns.

## Outputs
- CSV export of the current registry findings view.

## Notes
- Indicator coverage depends on available registry artifacts.
