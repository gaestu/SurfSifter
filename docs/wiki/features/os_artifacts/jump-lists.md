# Jump Lists (Subtab)

Jump Lists subtab - Windows jump list entries.

## Purpose
- Review recent and pinned jump list items.
- Filter by browser or pin status for analysis.

## Inputs
- Jump list tables in the evidence database.

## Filters and controls
- Filters: browser, pin status, and URLs-only toggle.
- Export CSV button.
- Table view with detail dialog on double-click.

## Outputs
- CSV export of the current jump list view.
- Detail dialog for a jump list entry.

## Notes
- Jump lists are Windows-specific artifacts.
