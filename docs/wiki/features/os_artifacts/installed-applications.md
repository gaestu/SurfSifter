# Installed Applications (Subtab)

Installed Applications subtab - detected software inventory.

## Purpose
- Review installed software and publishers.
- Filter to items of forensic interest.

## Inputs
- Installed software tables in the evidence database.

## Filters and controls
- Filters: search by name/publisher and "Forensic Interest Only".
- Export CSV button.
- Table view with detail dialog on double-click.

## Outputs
- CSV export of the current installed applications view.
- Detail dialog for a software entry.

## Notes
- Availability depends on extracted OS artifacts.
