# Grid (Subtab)

Grid subtab - thumbnail gallery of images.

## Purpose
- Quickly browse images as thumbnails.
- Select and tag images for export or reporting.

## Inputs
- Image records and files from the evidence database and case folder.

## Filters and controls
- Uses the main Images tab filters (tags, source, extension, size, hash match).
- Thumbnail grid with checkboxes, selection, and context menu.
- Actions: Export Selected, Tag Selected/Checked, Check Known Hashes, Find Similar, pagination.

## Outputs
- Exported image files saved to disk.
- Tags and hash match results stored in the case database.

## Notes
- Double-click opens a full preview dialog.
