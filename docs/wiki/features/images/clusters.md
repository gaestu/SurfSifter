# Clusters (Subtab)

Clusters subtab - perceptual hash clusters of similar images.

## Purpose
- Group visually similar images for quick triage.
- Review cluster members and export cluster lists.

## Inputs
- Image records with perceptual hashes from the evidence database.

## Filters and controls
- Uses the main Images tab filters (tags, source, extension, size, hash match).
- Cluster list with member detail view.
- Actions: Export Clusters CSV, Tag Selected, Check Known Hashes.

## Outputs
- Cluster CSV export saved to disk.
- Tags and hash match results stored in the case database.

## Notes
- Clustering quality depends on available perceptual hashes.
