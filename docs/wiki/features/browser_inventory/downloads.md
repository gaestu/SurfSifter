# Downloads (Subtab)

Browser Downloads subtab - parsed download history entries.

## Purpose
- Review browser download history and states.
- Tag and inspect downloads by filename or status.

## Inputs
- Browser downloads tables from the evidence database.

## Filters and controls
- Filters: browser, download state, and filename text.
- Apply button to run the current filters.
- Context menu for details, open/copy URL, and tagging.

## Outputs
- Tags stored on selected download records.
- Detail dialog for a single download.

## Notes
- This is browser download history, not the Downloads tab used to fetch files.
