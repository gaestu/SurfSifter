# Inventory (Subtab)

Inventory subtab - shows discovered browser artifact files and their status.

## Purpose
- Review raw browser databases and cache artifacts found during extraction.
- See extraction and ingestion status at a glance.

## Inputs
- Browser inventory records from the evidence database.
- Extracted artifact paths on disk.

## Filters and controls
- Filters: browser, artifact type, and status.
- Refresh button to reload the inventory list.
- Table context menu for details and file/path actions.

## Outputs
- Detail dialog for the selected artifact.
- Copy/open actions for artifact paths.

## Notes
- Re-ingest actions are handled in the Extraction tab.
