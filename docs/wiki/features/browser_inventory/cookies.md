# Cookies (Subtab)

Cookies subtab - parsed cookies with domain and browser context.

## Purpose
- Inspect cookie values and metadata across browsers.
- Tag cookies tied to specific domains or activity.

## Inputs
- Cookies tables from the evidence database.

## Filters and controls
- Filters: browser and domain text.
- Apply button to run the current filters.
- Context menu for details, copy values, and tagging.

## Outputs
- Tags stored on selected cookie records.
- Detail dialog for a single cookie.

## Notes
- Values may be encrypted or partially decoded depending on the source browser.
