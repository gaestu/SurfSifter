# Auth Tokens (Subtab)

Auth Tokens subtab - authentication tokens recovered from storage.

## Purpose
- Review tokens such as JWTs, OAuth, and API keys.
- Tag high-risk tokens or origins for follow-up.

## Inputs
- Auth token tables derived from web storage artifacts.

## Filters and controls
- Filters: token type, risk, origin text, and include expired.
- Apply button to run the current filters.
- Context menu for details, copy values, and tagging.

## Outputs
- Tags stored on selected token records.
- Detail dialog for an auth token entry.

## Notes
- Handle token values as sensitive data.
