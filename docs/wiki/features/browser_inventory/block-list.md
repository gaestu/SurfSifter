# Block List (Subtab)

Block List subtab - sites where autofill is disabled.

## Purpose
- Review sites the user explicitly blocked from autofill.
- Tag blocked sites that are relevant to the case.

## Inputs
- Autofill block list tables from the evidence database.

## Filters and controls
- Filters: browser and site/domain text.
- Apply button to run the current filters.
- Context menu for details and tagging.

## Outputs
- Tags stored on selected block list entries.
- Detail dialog for a blocked site record.

## Notes
- This subtab is Edge-specific.
