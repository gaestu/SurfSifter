# Storage Keys (Subtab)

Storage Keys subtab - LocalStorage and SessionStorage key-value entries.

## Purpose
- Review individual storage keys and values by origin.
- Tag keys of interest for follow-up.

## Inputs
- Storage key tables from the evidence database.

## Filters and controls
- Filters: origin, storage type (local/session), browser, and key name text.
- Apply button to run the current filters.
- Context menu for details, copy values, and tagging.

## Outputs
- Tags stored on selected storage key entries.
- Detail dialog for a storage key record.

## Notes
- Storage values may be large; use filters to narrow results.
