# Web Storage (Subtab)

Web Storage subtab - container for LocalStorage, SessionStorage, IndexedDB, and tokens.

## Purpose
- Review site storage data captured from browsers.
- Navigate to specific storage views (overview, keys, IndexedDB, tokens).

## Inputs
- Web storage and IndexedDB tables from the evidence database.

## Filters and controls
- Nested subtabs: Sites Overview, Storage Keys, IndexedDB, Auth Tokens.
- Each nested tab provides its own filters and context menu actions.

## Outputs
- Tags stored on selected storage artifacts.
- Detail dialogs for individual records.

## Notes
- Storage data can be large; use filters to narrow results.
