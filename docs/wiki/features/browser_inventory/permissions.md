# Permissions (Subtab)

Permissions subtab - site permission settings.

## Purpose
- Review per-site permission decisions (camera, location, notifications, etc.).
- Tag permission records tied to relevant domains.

## Inputs
- Site permissions tables from the evidence database.

## Filters and controls
- Filters: browser and permission type.
- Apply button to run the current filters.
- Context menu for details and tagging.

## Outputs
- Tags stored on selected permission records.
- Detail dialog for a permission entry.

## Notes
- Permission types vary by browser and version.
