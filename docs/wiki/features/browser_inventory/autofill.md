# Autofill (Subtab)

Autofill subtab - container for form data, logins, addresses, and related artifacts.

## Purpose
- Group autofill-related artifacts in one place.
- Navigate to the specific autofill data type you need.

## Inputs
- Autofill, credentials, and related tables from the evidence database.

## Filters and controls
- Nested subtabs: Form Data, Saved Logins, Addresses, Search Engines, Deleted History, Block List.
- Each nested tab provides its own filters and context menu actions.

## Outputs
- Tags stored on selected autofill artifacts.
- Detail dialogs for individual records.

## Notes
- Some nested tabs are browser-specific (e.g., Firefox deleted history, Edge block list).
