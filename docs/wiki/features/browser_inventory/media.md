# Media (Subtab)

Media subtab - media playback history.

## Purpose
- Review audio/video playback history captured by the browser.
- Tag media records for reporting.

## Inputs
- Media history tables from the evidence database.

## Filters and controls
- Filter: browser.
- Apply button to run the current filters.
- Context menu for details, open/copy URL, and tagging.

## Outputs
- Tags stored on selected media records.
- Detail dialog for a media entry.

## Notes
- Availability depends on browser support for media history.
