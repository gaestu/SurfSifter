# Sites Overview (Subtab)

Sites Overview subtab - per-site web storage summary.

## Purpose
- Summarize how many storage keys exist per site/origin.
- Identify high-storage sites quickly.

## Inputs
- Web storage summary tables from the evidence database.

## Filters and controls
- Filters: site/origin text and minimum key count.
- Apply button to run the current filters.
- Context menu for details, copy origin, and tagging.

## Outputs
- Tags stored on selected site summary rows.
- Detail dialog for a site summary entry.

## Notes
- Counts combine local and session storage for each origin.
