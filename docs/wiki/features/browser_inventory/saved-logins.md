# Saved Logins (Subtab)

Saved Logins subtab - stored website credentials.

## Purpose
- Review saved login entries and associated metadata.
- Tag credentials tied to key sites.

## Inputs
- Credentials tables from the evidence database.

## Filters and controls
- Filters: browser and origin URL text.
- Apply button to run the current filters.
- Context menu for details and tagging.

## Outputs
- Tags stored on selected login entries.
- Detail dialog for a credential record.

## Notes
- Password values may be masked or encrypted depending on browser support.
