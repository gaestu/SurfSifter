# Restored Tabs (Subtab)

Restored Tabs subtab - browser session restore data.

## Purpose
- Review tabs and windows that were open at last browser shutdown.
- Tag session entries tied to specific activity.

## Inputs
- Session restore tables from the evidence database.

## Filters and controls
- Filter: browser.
- Apply button to run the current filters.
- Context menu for details, open/copy URL, and tagging.

## Outputs
- Tags stored on selected session records.
- Detail dialog for a session entry.

## Notes
- Session restore data reflects the last saved browser state.
