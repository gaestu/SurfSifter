# Bookmarks (Subtab)

Bookmarks subtab - parsed bookmarks and folders.

## Purpose
- Review saved bookmarks by browser and folder.
- Tag bookmarks for reporting or grouping.

## Inputs
- Bookmarks tables from the evidence database.

## Filters and controls
- Filters: browser and bookmark folder.
- Apply button to run the current filters.
- Context menu for details, open/copy URL, and tagging.

## Outputs
- Tags stored on selected bookmark records.
- Detail dialog for a single bookmark.

## Notes
- Folder names come from the browser's bookmark hierarchy.
