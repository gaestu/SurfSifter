# Search Engines (Subtab)

Search Engines subtab - configured browser search providers.

## Purpose
- Review installed search engines and keywords.
- Tag providers relevant to the investigation.

## Inputs
- Search engine tables from the evidence database.

## Filters and controls
- Filters: browser and keyword text.
- Apply button to run the current filters.
- Context menu for details and tagging.

## Outputs
- Tags stored on selected search engine records.
- Detail dialog for a search engine entry.

## Notes
- Entries reflect browser configuration at extraction time.
