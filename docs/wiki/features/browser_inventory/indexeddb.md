# IndexedDB (Subtab)

IndexedDB subtab - IndexedDB databases and records.

## Purpose
- Review IndexedDB entries and metadata by origin and database.
- Tag relevant records for analysis.

## Inputs
- IndexedDB tables from the evidence database.

## Filters and controls
- Filters: origin, database name, and object store.
- Apply button to run the current filters.
- Context menu for details and tagging.

## Outputs
- Tags stored on selected IndexedDB records.
- Detail dialog for an IndexedDB entry.

## Notes
- IndexedDB schemas vary widely by site and browser.
