# Deleted History (Subtab)

Deleted History subtab - deleted form history entries (Firefox).

## Purpose
- Review form history items that were deleted by the user.
- Tag deleted entries for investigative context.

## Inputs
- Deleted form history tables from the evidence database.

## Filters and controls
- Filter: browser.
- Apply button to run the current filters.
- Context menu for details and tagging.

## Outputs
- Tags stored on selected deleted history entries.
- Detail dialog for a deleted record.

## Notes
- This subtab is Firefox-specific.
