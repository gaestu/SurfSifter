# Addresses (Subtab)

Addresses subtab - structured autofill profile data.

## Purpose
- Review saved address/contact fields stored by the browser.
- Tag specific profile fields for analysis.

## Inputs
- Autofill profile token tables from the evidence database.

## Filters and controls
- Filters: browser and field type text.
- Apply button to run the current filters.
- Context menu for details and tagging.

## Outputs
- Tags stored on selected address token entries.
- Detail dialog for a profile token record.

## Notes
- Chromium-based browsers store address fields as tokens per profile.
