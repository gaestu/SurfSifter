# Extensions (Subtab)

Extensions subtab - installed browser extensions.

## Purpose
- Review installed extensions and their metadata.
- Identify high-risk or suspicious extensions and tag them.

## Inputs
- Extensions tables from the evidence database.

## Filters and controls
- Filters: browser, category, and minimum risk score.
- Apply button to run the current filters.
- Context menu for details, copy extension ID, open homepage, and tagging.

## Outputs
- Tags stored on selected extension records.
- Detail dialog for an extension entry.

## Notes
- Risk scores are derived from extension metadata when available.
