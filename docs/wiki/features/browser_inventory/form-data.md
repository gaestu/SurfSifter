# Form Data (Subtab)

Form Data subtab - traditional form field autofill entries.

## Purpose
- Review saved form field values captured by the browser.
- Tag specific field values for analysis.

## Inputs
- Autofill form data tables from the evidence database.

## Filters and controls
- Filters: browser and field name text.
- Apply button to run the current filters.
- Context menu for details and tagging.

## Outputs
- Tags stored on selected form data entries.
- Detail dialog for a form entry.

## Notes
- Field names and values reflect what the browser saved at the time.
