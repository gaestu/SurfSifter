# Cache (Subtab)

Cache subtab - browser cache entries.

## Purpose
- Review cached resources captured from browser caches.
- Filter by domain or content type for targeted analysis.

## Inputs
- Cache entries tables from the evidence database.

## Filters and controls
- Filters: browser, domain, and content type.
- Apply button plus pagination controls and page size.
- Context menu for details, open/copy URL, and tagging.

## Outputs
- Tags stored on selected cache entries.
- Detail dialog for a cache record.

## Notes
- Cache views are paginated to handle large datasets.
