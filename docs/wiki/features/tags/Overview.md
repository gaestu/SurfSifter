# Tags (Tab)

Manage the tag list used across all artifacts in the case.

## Purpose
- Create, rename, merge, and delete tags.
- Review tag usage counts to keep tagging consistent.

## When to use
- When you want to standardize or clean up tags applied to artifacts.
- When you need to add new tags before tagging more items.

## Data sources
- Case database tag tables and tag usage counts.

## Key controls
- Search box to filter tag list.
- Actions: Create Tag, Merge Selected, Delete Selected.
- Rename via double-click or context menu.

## Outputs
- Tag changes saved to the case database (applies across artifacts).
- Merge/delete operations update tag assignments on existing artifacts.

## Subtabs
- None

## Notes
- Deleting a tag removes it from all artifacts.
