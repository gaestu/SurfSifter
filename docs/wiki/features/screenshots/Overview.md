# Screenshots (Tab)

Screenshots Tab Widget - v1.13.0

## Purpose
- Manage evidence screenshots for documentation and reporting.
- Add and review screenshot metadata with thumbnails.

## When to use
- When you need to capture, upload, or curate screenshots for reports.
- When you want to edit screenshot titles/captions or organize sequences.

## Data sources
- Evidence database screenshots table.
- Evidence folder on disk (stored screenshot files and thumbnails).

## Key controls
- Filters: sequence and source (sandbox vs upload).
- Actions: Upload, Delete Selected, Select All/Deselect All.
- Table view with thumbnails; double-click a row to edit metadata.

## Outputs
- Screenshot files saved under the evidence folder.
- Screenshot metadata (title, caption, sequence, hashes) stored in the evidence database.

## Subtabs
- None

## Notes
- Top-level tab for managing investigator-captured screenshots for forensic documentation.
