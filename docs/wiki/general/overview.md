# General Overview

SurfSifter is a cross-platform forensic workstation for analyzing browser artifacts, cached media, and OS evidence from disk images. It focuses on read-only, case-based workflows so evidence is never modified.

## Scope
- Offline extraction and analysis of browser artifacts, media, and selected OS artifacts
- Correlation and reporting through a native GUI
- Designed primarily for Windows evidence on any host OS

## Supported platforms
- **Host OS:** Linux, macOS, Windows
- **Evidence:** E01/EWF disk images (multi-partition supported)

## Status
- **Beta:** core functionality is feature-complete and actively tested
