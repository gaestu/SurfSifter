# Tagged File List (Report Module)

Shows tagged file list entries with optional match and deletion filters.

## Purpose
- Provide a report-ready list of relevant files.
- Filter by tags or reference list matches.

## Inputs
- File list entries generated during extraction or file list import.
- Tags and reference list matches applied to files.

## Filters and controls
- Show Title: Toggle section title.
- Limit: Maximum number of files to display.
- Tags: Filter by tag (All, Any Tag, or a specific tag).
- Matches: Filter by reference list matches (All, Any Match, or a specific list).
- Include Deleted Files: Include files flagged as deleted.
- Sort By: Order by modified time, name, or path.
- Show Filter Info: Display selected filters below the table.

## Output
- Table of file path, size, timestamps, and deletion status.

## Notes
- Match options appear only when reference list matches exist.
