# Activity Summary (Report Module)

Summarizes timeline activity with daily density and inactivity gaps.

## Purpose
- Provide a high-level view of when the system was active.
- Highlight periods of inactivity and overall activity volume.

## Inputs
- Timeline events generated by extractors and timeline processing.

## Filters and controls
- Event Types: All Events, Browser Activity, Downloads, Authentication, Media Playback.
- Minimum Gap (hours): Threshold for reporting inactivity gaps.
- Show Daily Breakdown: Toggle the per-day activity visualization.
- Max Days to Show: Limit how many days appear in the daily breakdown.
- Show Event Type Breakdown: Toggle the event category table.
- Minimum Confidence: Filter events by confidence level.

## Output
- Activity period, total events, and average events per day.
- Daily activity chart (optional).
- Significant inactivity gaps (optional).
- Event-type breakdown table (optional).

## Notes
- Requires timeline data; if none exists, the module shows a no-events message.
- Confidence filtering hides events below the selected level.
