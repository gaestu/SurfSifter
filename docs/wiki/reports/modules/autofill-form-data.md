# Autofill Form Data (Report Module)

Groups autofill entries by domain or URL and presents them as per-site tables.

## Purpose
- Provide a domain-focused view of saved autofill data.
- Present form entries in a readable, non-technical layout.

## Inputs
- Autofill artifacts extracted from browser profiles.
- Tags applied to autofill entries.

## Filters and controls
- Show Title: Toggle module title.
- Show Description: Include explanatory text for non-technical readers.
- Tags: Multi-select tag filter.
- Group by Domain: Group entries by domain/URL.
- Sort By: Order entries within each domain.
- Show Filter Info: Display selected filters below the content.

## Output
- Domain sections with form field name, value, and usage dates.
- Totals for domains and entries.

## Notes
- Entries without a recognizable domain are grouped separately.
