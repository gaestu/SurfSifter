# System Summary (Windows) (Report Module)

Summarizes Windows system information extracted from registry artifacts.

## Purpose
- Provide a high-level overview of OS configuration and user context.
- Capture installed software, autostart entries, and network configuration.

## Inputs
- OS indicators generated by the Windows registry extractor.

## Filters and controls
- Sections to Include: Select which sections to show (system info, users, software, autostart, network).
- Show System Profiles: Include system accounts such as LocalService or NetworkService.
- Software List Limit: Limit the number of installed software entries.
- Show Registry Paths: Include registry path details for each indicator.

## Output
- System information table (OS version, install date, timezone, etc.).
- User accounts table.
- Installed software list (optional).
- Autostart entries list (optional).
- Network configuration summary (optional).

## Notes
- Windows-only; if registry artifacts are not extracted, the module will be empty.
