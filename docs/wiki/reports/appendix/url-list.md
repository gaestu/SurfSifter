# URL List (Appendix Module)

Lists URLs as appendix content, optionally grouped by domain.

## Purpose
- Provide a complete URL list as supporting evidence.
- Filter by tags or reference list matches.

## Inputs
- URLs extracted from browser artifacts.
- URL tags and reference list matches.

## Filters and controls
- Tags: Multi-select tag filter.
- Matches: Multi-select reference list filter.
- Filter Mode: OR (any tag or match) or AND (must have both).
- Group by Domain (FQDN): Group URLs under their domain.

## Output
- URL list, grouped or ungrouped, with a total count.

## Notes
- Use the appendix when the full list is too long for a main section.
