# File List (Appendix Module)

Lists files in a compact appendix layout with tag and match filters.

## Purpose
- Provide a complete list of relevant files as supporting evidence.
- Filter by tags or reference list matches.

## Inputs
- File list entries and reference list matches.
- Tags applied to file list entries.

## Filters and controls
- Tags: Multi-select tag filter.
- Matches: Multi-select reference list filter.
- Filter Mode: OR (any tag or match) or AND (must have both).
- Include Deleted Files: Include files flagged as deleted.

## Output
- Compact file list with path, timestamps, and size information.

## Notes
- Use this appendix when the full file list would overwhelm the main report sections.
